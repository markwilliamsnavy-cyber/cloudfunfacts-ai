# CloudFunFacts AI

CloudFunFacts AI is a serverless AWS application that retrieves cloud computing facts from DynamoDB and uses Amazon Bedrock Claude 4.1 to generate engaging AI-enhanced responses.

## Architecture

Amplify Frontend → API Gateway → Lambda → DynamoDB → Amazon Bedrock

## AWS Services Used

* AWS Amplify
* Amazon API Gateway
* AWS Lambda
* Amazon DynamoDB
* Amazon Bedrock Claude 4.1

## Features

* Random cloud facts
* AI-generated explanations
* Serverless architecture
* Web frontend hosted on Amplify
