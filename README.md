# CloudFunFacts AI

An AI-powered serverless AWS application that retrieves cloud computing facts from DynamoDB and uses Amazon Bedrock Claude 4.1 to generate engaging and educational cloud facts.

## Live Demo

https://staging.d3a5cvcl597uzx.amplifyapp.com

## GitHub Repository

https://github.com/markwilliamsnavy-cyber/cloudfunfacts-ai

## Architecture

```text
AWS Amplify
      ↓
Amazon API Gateway
      ↓
AWS Lambda (Python)
      ↓
Amazon DynamoDB
      ↓
Amazon Bedrock Claude 4.1
```

## AWS Services Used

* AWS Amplify
* Amazon API Gateway
* AWS Lambda
* Amazon DynamoDB
* Amazon Bedrock
* Amazon CloudWatch

## Features

* Retrieves random cloud computing facts
* Uses Amazon Bedrock Claude 4.1 to generate engaging explanations
* Serverless architecture
* Responsive web interface hosted on AWS Amplify
* REST API integration through API Gateway

## Skills Demonstrated

* Python
* AWS Lambda
* API Gateway
* DynamoDB
* Amazon Bedrock
* Serverless Architecture
* REST APIs
* Cloud Troubleshooting
* CORS Configuration
* Frontend Deployment with Amplify

## Project Outcome

Built and deployed a production-ready serverless web application that integrates generative AI with AWS cloud services. The application retrieves cloud computing facts from DynamoDB, enhances them using Amazon Bedrock Claude 4.1, and presents them through a web interface hosted on AWS Amplify.
