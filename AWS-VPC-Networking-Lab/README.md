# AWS VPC Networking Lab

## Project Overview

Designed and deployed a custom AWS Virtual Private Cloud (VPC) to demonstrate networking, security, and infrastructure concepts commonly used in enterprise cloud environments.

## Architecture

Internet
↓
Internet Gateway
↓
Public Subnet (10.0.1.0/24)
↓
Public EC2 Web Server (Apache)

Private Subnet (10.0.2.0/24)
↓
Private EC2 Application Server

## Components

* Custom VPC (10.0.0.0/16)
* Public Subnet
* Private Subnet
* Internet Gateway
* Public Route Table
* Private Route Table
* Security Groups
* Public EC2 Instance
* Private EC2 Instance
* Apache Web Server

## Skills Demonstrated

* AWS VPC Design
* Subnet Planning
* Route Tables
* Internet Gateway Configuration
* Security Groups
* Linux Administration
* SSH Connectivity
* Apache Web Server Deployment
* Cloud Networking
* Infrastructure Troubleshooting

## Key Achievements

* Built a custom VPC from scratch
* Configured public and private subnets
* Implemented secure access using Security Groups
* Connected to EC2 instances using SSH key pairs
* Deployed an Apache web server
* Validated internet connectivity through a public subnet
* Demonstrated private subnet isolation by deploying a server without a public IP

## Outcome

Successfully created a multi-tier AWS networking environment demonstrating public-facing and private infrastructure components following cloud networking best practices.
